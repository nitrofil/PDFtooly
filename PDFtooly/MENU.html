- PDFtooly : divers outils pour faciliter la gestion des fichiers PDF.


- Suivant l'option choisie, maintenir CTRL pour séléctionner plusieurs fichiers.

- Tous les fichiers modifiés, convertis, divisés ou autres, se trouvent dans le dossier "Téléchargement".


- Une connexion internet est requise pour charger les librairies d'usages :

jspdf.umd.min.js, pdf-lib, jszip.min.js, pdf.min.js, FileSaver.min.js, jszip.min.js, pdf.worker.min.js, pdfjs-dist.


- Tous les fichiers sont traités localement, rien n'est envoyer en dehors de l'ordinateur.
